# Data Analytics
Репозиторий с проектами по анализу данных и машинному обучению

## О репозитории
Этот репозиторий создан для демонстрации моих навыков в области **аналитики данных**. Здесь представлены проекты, выполненные в ходе обучения и практической работы с реальными и открытыми наборами данных.

## student_data_analysis.ipynb
Данный проект посвящен исследовательскому анализу данных (EDA) и построению моделей машинного обучения для прогнозирования академической успеваемости студентов.
